Operadores Logicos

Operador	Operação
AND		Duas condições sejam verdadeiras
OR		Pelo menos uma condição seja verdadeira

Comandos condicionais ou Estruturas Condicionais

if - Realiza Testes condicionais, executa um bloco SE de uma determinada condição for atendida, avalia se condição é verdadeira ou não.

Bloco - Trecho de código que depende de outro trecho

else - é executada caso a condição do comando if não seja atendida

elif - caso haja necessidade de mais condições entre o if e o else (sempre a primeira condição verdadeira que encontrar).
